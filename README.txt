To calculate values to plot activity-density x thickness graphs: 

part3.py      : Creates a list containing all of the appropriate data
part5.py      : Calculates the detected gamma activity and its error
part6.py      : Calculates the materials' density
part8.py      : Calculates the density multiplied by the thickness
part7.py      : Calculates the average density of a material- used for materials tested with non-uniform layers
part8b.py     : Creates a list containing some of the approritae data for a combination of materials
part_table.py : Creates a table in a text file containg vital input data and code calulcated values
part8c.py     : Creates a list containing all of the appropriate data for a material/ combination of materials
function.py   : The line of best fit the graphs should show- graphs not plotted in this code (in QtiPlot)
part4.py      : Uses the classes from other files to plot a variety of graphs when called
part4b.py     : Creates a table containig all the relevant data for each material/combination
testing.py    : Uses all the previous files to produce a final table and save it to a location
